Diseño web zx auto

Se ha desarrollado un sitio web para Salcedo Vehículos, una concesionaria en Perú que se especializa en la marca ZX AUTO. El objetivo principal es impulsar las ventas de sus camionetas TERRALORD en el mercado peruano y fortalecer la reputación de la marca destacando sus atributos clave.

Desafio

Superar la mala fama de las Pick-ups chinas y destacar en un mercado competitivo.

Propuesta

Usar la web para resaltar la calidad, experiencia y atributos únicos de la TERRALORD, apoyándose en la imagen de Ramón Ferreyros y una narrativa de aventura y versatilidad.

UI Kit

Se creó un UI kit con el propósito de agilizar y estandarizar la generación de pantallas. En este kit se definieron tamaños de texto, paletas de colores y otros elementos como cards. Además, se incluyeron componentes reutilizables, iconografía y estilos tipográficos para asegurar una apariencia visual coherente y profesional en todo el sitio.
